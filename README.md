# Air Quality Monitoring System using ESP32

## Overview
This project monitors air quality using an MQ135 gas sensor and ESP32. It helps detect pollution levels and harmful gases in the environment.

## Features
- Air Quality Monitoring
- Pollution Detection
- Real-Time Sensor Readings
- IoT Ready

## Components
- ESP32
- MQ135 Sensor
- Breadboard
- Jumper Wires

## Working
The MQ135 sensor detects harmful gases and provides analog output. ESP32 reads the values and determines air quality levels.
